The project presents a simulation model and software implementation of an intelligent automation system that responds to both ambient light intensity and time parameters. The shortcomings of classic photocell systems (for example, incorrect switching due to transient shadow effects during the day) cause significant energy loss. The project aims to eliminate this problem by designing a microcontroller-controlled system supplemented with a software real-time clock (RTC) and logic gating, which operates in an event-driven, asynchronous manner.
